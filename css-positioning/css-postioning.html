CSS Positioning

static relative absolute Fixed sticky

Position static

Default

, follows normal document flow

Position relative :

Position relative to its normal position. Can set position with properties :

  • left
  • right
  • top
  • bottom

z-index :

  • Specify priority of an element in relative position
  • heigher priority element can overlap on lower elements
  • low-high : 1 - inifinity priority

Example :

red : static . black : relative, z-index 5. blue : relative, z-index 2.

Position absolute :

Position relative to its nearest positioned parent element (ancestor)

Position fixed :

Position relative to the viewport remains fixed during scrolling
