CSS Positioning

static relative absolute Fixed sticky

Position relative :

Position relative to its normal position. Can set position with properties :

  • left
  • right
  • top
  • bottom

z-index :

  • Specify priority of an element in relative position
  • heigher priority element can overlap on lower elements
  • low-high : 1 - inifinity priority

Example :

  • red : static.
  • black : relative, z-index 5, left: 30px; top: 40px;
  • blue : relative, z-index 2.

Position absolute :

Position relative to its nearest positioned parent element (ancestor). If parent element is static then the absolute value means the whole screen. (eg: top: 100px [it start from the top of the screen.])

Examples :

  1. Parent as screen (parent's position: static)
